## Käyttöohje

1. Kloonaa tämä repo koneellesi

```bash
git clone <repo_url>
```

2. Luo virtuaaliympäristö ja asenna tarvittavat kirjastot

```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux
source venv/bin/activate
```

```bash
pip install -r requirements.txt
```
